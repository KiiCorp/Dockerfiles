# Getting Started
## MacOSX

Install homebrew.
```
ruby -e "$(curl -fsSL https://raw.github.com/Homebrew/homebrew/go/install)"
```

Install boot2docker with brew.
```
brew bundle
```

An environment variable.
```
export DOCKER_HOST=tcp://127.0.0.1:4243
```
